"저는 웹 사이트를 만들어내는 과정 속에서

디자인과 코드를 조화롭게 결합하는 작업이

제 창의성과 논리적 사고를 모두 발휘할 수 있어서

재밌습니다.

트렌드를 따라가며 꾸준히 공부하여

TECH STACKS

아이콘에 마우스를 올리거나 탭하면 자세한 설명이 나옵니다.

HTML5

웹 접근성과 웹 표준 준수에 중점을 두고, 시맨틱 태그를 활용하여 웹 페이지의 구조를 설계할 수 있습니다.

CSS3

position, display 속성과 미디어 쿼리를 활용해 유연한 반응형 웹 디자인을 구현할 수 있습니다.

JAVASCRIPT

ES6+의 문법을 이해하며, 배열 처리, DOM 조작, 이벤트 핸들링 등을 통한 개발을 할 수 있습니다.

JQERY

DOM 조작과 이벤트 관리를 위한 jQuery의 메서드를 효과적으로 활용할 수 있습니다.

REACT

React구조와 함수형 컴포넌트, Hook을 잘 활용할 수 있습니다.

SASS

SCSS의 변수, 믹스인, 함수 등을 적극적으로 활용하여 스타일 작업을 원활히할 수 있습니다.

mongoDB

Mongoose 라이브러리를 통해 스키마를 설계하고 데이터를 관리할 수 있습니다.

TYPESCRIPT

TypeScript의 기본 문법과 타입 시스템을 이해하며, 인터페이스와 제네릭을 활용해 타입 안정성을 높이고 코드를 구조화할 수 있습니다.

FIGMA

사용자 입장을 고려한 UI를 만들고, 디자인 과정을 원활하게 진행할 수 있습니다.

PHOTOSHOP

마스크효과, 레이어 활용 등을 이용해 다양한 그래픽 요소를 제작할 수 있습니다.

ILLUSTRATOR

펜툴 활용과 기타도구들을 이용해서 벡터 형식의 디자인 요소를 제작할 수 있습니다.

GITHUB

commit, push, pull, clone 등 git 명령어를 사용하여 레파지토리에 접근할 수 있습니다.
